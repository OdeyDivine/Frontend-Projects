# 🎨 Frontend Practice

This repository contains small frontend projects I built while learning and revising **HTML, CSS, and JavaScript**.

These are practice projects — not perfect, but proof of progress. Each one helped me understand layout, styling, and responsive design a little better and optimised.

---

## 📁 Projects Included

| Project | Description |
|---------|-------------|
| `Facebook_Landing_Page` | A clone of the Facebook login/signup page |
| `Linkedin_Landing_Page` | LinkedIn-style landing page layout |
| `base_apparel` | Desktop view practice for a fashion brand |
| `product_preview` | Product card with laptop view styling |
| `profile-card` | A simple profile card component |
| `qrcode` | QR code component with favicon added |
| `restaurant_web` | Practice layout for a restaurant website |
| `DotCampus_HTML` | Form and table practice |

> *More projects will be added as I learn.*

---

## 🛠️ What I'm Practicing

- HTML5 (semantic tags, forms, tables)
- CSS3 (Flexbox, Grid, positioning)
- Responsive design (mobile, tablet, desktop)
- Basic UI components (cards, buttons, forms)

---

## 📌 How to View

1. Clone the repo  
   `git clone https://github.com/OdeyDivine/frontend-practice.git`

2. Open any project folder  
   `cd Facebook_Landing_Page`

3. Open `index.html` in your browser
